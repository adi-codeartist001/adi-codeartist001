## About Me

I'm a second-year B.Tech (AI & Data Analytics) student who ships real product, not just tutorials. I'm currently the **sole frontend architect** on **PipeBit HRM**, a SaaS HR platform for Indian startups — owning everything from role-based auth flows to payroll UI to bulk data import. Alongside that, I build AI-driven automation systems at **Snyco** and contribute actively to open source through **GSSoC 2026**.

I like picking projects that force me to learn something painful and useful — the kind of stuff you can't fake in an interview.

- 🔭 Building the admin panel for a **WhatsApp Business automation platform** (Next.js)
- 🧠 Architected the entire **PipeBit HRM** frontend (React + Vite + Tailwind + shadcn/ui) — auth, payroll, attendance, leave management, RBAC
- ⚙️ Comfortable across the stack: React/Next.js on the frontend, FastAPI/Node on the backend, PostgreSQL/MongoDB for data
- 🏆 10+ hackathons, finalist at **Vihaan 9.0 (DTU)** and **Health Hack 2026 (IIT Bhopal × Johns Hopkins)**
- 🌍 Open source contributor — GSSoC 2026
- 📊 Member, Microsoft UP Power BI Community
